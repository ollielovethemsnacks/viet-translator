# Install Viet Translator with AltStore

## What is AltStore?

AltStore is a free, open-source app installer for iOS that lets you sideload apps without jailbreaking or an Apple Developer account.

## Prerequisites

1. **Windows, macOS, or Linux computer**
2. **iPhone with iOS 14+**
3. **Lightning or USB-C cable**
4. **Apple ID** (free account works)

---

## Step 1: Install AltStore

### Windows
1. Download AltServer from: https://altstore.io/
2. Install iTunes from Microsoft Store (not the web version)
3. Install iCloud from Microsoft Store
4. Run AltServer

### macOS
1. Download AltServer from: https://altstore.io/
2. Move to Applications folder
3. Run AltServer

### Linux
1. Use AltLinux or Wine with AltServer
2. Or use a VM with Windows/macOS

---

## Step 2: Install AltStore on Your iPhone

1. Connect iPhone to computer via USB
2. Open AltServer on computer
3. Click AltServer icon in system tray/menu bar
4. Select **"Install AltStore"** → Choose your iPhone
5. Enter your Apple ID when prompted
6. AltStore will install on your iPhone

---

## Step 3: Trust AltStore

On your iPhone:
1. Open **Settings** → **General** → **VPN & Device Management**
2. Find your Apple ID under "Developer App"
3. Tap **Trust**

---

## Step 4: Download Viet Translator IPA

1. Go to: https://github.com/ollielovethemsnacks/viet-translator/releases
2. Download the latest **VietTranslator.ipa** file
3. Save to your iPhone (Files app)

---

## Step 5: Install Viet Translator

1. Open **AltStore** on your iPhone
2. Tap **"+"** button (top left)
3. Select **VietTranslator.ipa** from Files
4. Wait for installation
5. Viet Translator will appear on home screen

---

## Step 6: Keep AltStore Running

**Important:** AltStore apps expire after 7 days. To refresh:

1. Connect iPhone to computer with AltServer running
2. Open AltStore on iPhone
3. Apps will auto-refresh

**Or enable automatic refresh:**
1. AltStore → Settings → Enable "Background Refresh"
2. Connect to same WiFi as computer

---

## Troubleshooting

### "Cannot Connect to AltServer"
- Make sure iPhone and computer are on same WiFi
- Try USB connection instead
- Restart AltServer

### "App Not Trusted"
- Go to Settings → General → VPN & Device Management
- Trust your Apple ID

### "Installation Failed"
- Make sure you have enough storage
- Try restarting iPhone
- Re-download the IPA file

### App Won't Open
- Check iOS version (needs iOS 14+)
- Re-install with AltStore
- Check for iOS updates

---

## Alternative: AltStore Source

You can add Viet Translator as a source in AltStore:

1. Open AltStore
2. Go to **Sources** tab
3. Tap **"+"**
4. Enter: `https://ollielovethemsnacks.github.io/viet-translator/altstore.json`
5. Viet Translator will appear in Browse tab
